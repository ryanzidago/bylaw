# Bylaw.Ecto.Query

Ecto query validation APIs and checks for Bylaw.

This package exposes modules under the `Bylaw.Ecto.Query` namespace and depends
on `:ecto`.

## Installation

Add `bylaw_ecto_query` to your dependencies:

```elixir
def deps do
  [
    {:bylaw_ecto_query, "~> 0.1.0"}
  ]
end
```

Use the HexDocs guide for `prepare_query/3` wiring, the built-in check list,
check options, and issue metadata.
